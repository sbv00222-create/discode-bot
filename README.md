# Discord Link Only Bot

디스코드 채널에서 **링크가 포함되지 않은 메시지를 자동으로 삭제**해주는 봇입니다.

특정 채널을 링크 공유 전용 채널로 운영할 때 사용할 수 있습니다.

## 주요 기능

* 링크가 없는 일반 메시지 자동 삭제
* `http://`, `https://` 링크 감지
* 디스코드 초대 링크, 웹사이트 링크 등 URL 포함 메시지 허용
* 링크 전용 채널 관리에 적합
* 불필요한 채팅 방지

## 사용 목적

이 봇은 디스코드 채널을 링크 공유 전용으로 유지하기 위해 만들어졌습니다.

예를 들어 아래와 같은 메시지는 허용됩니다.

```txt
https://example.com
이 영상 봐주세요 https://youtube.com/example
```

아래와 같은 메시지는 자동 삭제됩니다.

```txt
안녕하세요
이거 확인해주세요
링크 어디에 있나요?
```

## 동작 방식

사용자가 메시지를 보내면 봇이 메시지 내용을 확인합니다.

메시지 안에 링크가 포함되어 있으면 그대로 유지하고, 링크가 없으면 해당 메시지를 자동으로 삭제합니다.

## 필요한 권한

봇이 정상적으로 작동하려면 다음 권한이 필요합니다.

* 메시지 보기
* 메시지 관리
* 메시지 기록 보기

특히 **메시지 관리 권한**이 없으면 메시지를 삭제할 수 없습니다.

## 주의사항

* 봇보다 권한이 높은 사용자의 메시지는 삭제하지 못할 수 있습니다.
* 관리자 권한을 가진 사용자의 메시지는 설정에 따라 삭제되지 않을 수 있습니다.
* 링크 감지 기준은 코드에서 수정할 수 있습니다.

## 사용 예시

링크 전용 채널에서 링크가 없는 메시지가 작성되면 봇이 자동으로 삭제합니다.

```txt
사용자: 안녕하세요
봇: 해당 메시지 삭제
```

```txt
사용자: https://github.com/example
봇: 메시지 유지
```
